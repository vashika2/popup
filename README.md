### 🎉 Popup — Personalised Surprise Messages

A lightweight, shareable web popup that lets you send someone
a personalised surprise message for any occasion.

Just build the link, send it, and they open it to a beautiful
animated popup made just for them.

#### ✨ Features

- Works for any occasion — birthday, congrats, love, thank you, good luck, welcome
- Custom name and message via URL parameters
- Falling confetti animation
- Floating sparkles over the card
- Slide-up message ribbon
- Fully responsive — looks great on mobile and desktop
- Built with plain HTML, CSS, and JavaScript — no frameworks

### 🔗 How to use

Send someone a link like this:

your-site.netlify.app?name=Priya&occasion=birthday&msg=Hope+your+day+is+magical!

Supported occasions: birthday · congrats · love · thankyou · goodluck · welcome

### 🛠️ Built with

- HTML / CSS / JavaScript
- Google Fonts — Pacifico + Nunito
- Canva — card artwork and design
